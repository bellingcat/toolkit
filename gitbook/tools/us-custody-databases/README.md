## Description

Custody information is public available and maintained in databases by each individual state. These databases allow you to see information such as:

* Full name of adult in custody
* Facility name
* Sentence start date / institution admission date
* Earliest release / parole date
* Demographic information
* List of offenses and case number
* Aliases
