<summary>DISTINCT FEATURES <mark style="color:$success;">(CLICK TO EXPAND)</mark></summary>



| Function                         | Soar Stoary                                                                        | Datawrapper Locator Map                                       |
| -------------------------------- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| Imagery integration              | Uses real satellite and user-uploaded maps from Soar Atlas in addition to basemaps | No imagery integration — only stylized basemaps               |
| Page-based narrative system      | Multi-page “Stoary” with map position lock + title + description per page          | Single-map layout only                                        |
| Map layers / imagery sources     | Pulls imagery layers directly from Soar Atlas                                      | Static vector basemap only                                    |
| Drawing tools                    | Built-in drawing and measurement tools (area, distance, KML import, annotations)   | No drawing tools — markers only                               |
| Measurement tools                | Distance and area in km²                                                           | None                                                          |
| KML / file import                | Yes (overlay external KMLs for AOI definition)                                     | CSV import only for coordinates                               |
| Story sequencing / page ordering | Yes (move page up/down; each page = unique view)                                   | No sequencing — static view, single page                      |
| Show / filter satellite paths    | Yes (filter imagery capture paths)                                                 | No satellite metadata at all                                  |
| Export format                    | Online share; no CSV export (visual focus)                                         | Exports as GeoJSON, image, or embed as iframe or embed as URL |
| Map projection choices           | Default Soar Atlas projections                                                     | Multiple projection options (e.g., globe)                     |
| User editing scope               | Deep map control (layers, views, page-level management)                            | Limited to marker placement and visual design                 |

</details>
