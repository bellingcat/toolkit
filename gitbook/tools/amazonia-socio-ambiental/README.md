# Amazonia Socio Ambiental (RAISG)

## URL

[https://www.amazoniasocioambiental.org/en/](https://www.amazoniasocioambiental.org/en/https://www.amazoniasocioambiental.org/en/)

## Description

Amazonia Socio Ambiental is the platform for RAISG, the Amazon Network of Georeferenced Socio-Environmental Information. The site hosts RAISG's statistical and geospatial datasets along with publications regarding deforestation, indigenous territory, and environmental policies in the region.

RAISG provides geospatial datasets related to protected natural areas and indigenous territories, projects and concessions threatening the Amazon, and deforestation. RAISG's online map of geospatial data is hosted by ArcGIS, and the site provides links to download datasets in shapefile and API format.

This data would be useful for researchers investigating land use or land exploitation in Bolivia, Brazil, Colombia, Ecuador, Guyanas and Suriname. For example, researchers could compare deforestation data over time to analyze land conservation efforts in the Amazon.

<figure><img src=".gitbook/assets/Screenshot 2025-12-02 at 9.41.04 PM.png" alt=""><figcaption></figcaption></figure>

Using RAISG's online map, researchers can compare assessed forest cover in 2000 (in green), deforestation that occurred between 2001-2010 (red/orange points), and Indigenous territory (light green/orange overlay). Users can also compare data related to protected natural areas, deforestation-related activities, and hotspots in the last 24 hours.&#x20;

Users can customize the appearance of the data visualizations by adjusting the layers and transparency of each. For additional customization, users can download individual files for use in their own models.

## Requirements

The site is available in English, Spanish, and Portuguese. No login or API is required.

## Limitations

Many of RAISG's articles are only available in Spanish.&#x20;

Using the datasets requires some knowledge of GIS tools to fully manipulate the data. Additionally, the online map is hosted using ArcGIS, which requires users to have some pre-existing knowledge of that tool as well.

## Ethical Considerations

RAISG is financed by several nonprofits from the Global North and the government foreign aid ministries of Norway and Sweden. It is unclear whether RAISG's methodology involved the consultation of indigenous communities in Amazonia, although two of the eight civil society organizations that integrate RAISG explicitly mention their work in partnership with Indigenous communities and organizations.
